## **Technology Stack**

- **Front-End:**
  React, TypeScript

- **Back-End:**

  - Node.js
  - Nim
  - GraphQL for handling communication between front-end and back-end

- **Database:**

  - SQL to store user information, website data, templates, etc.

- **File Storage:**

  - Linode

- **Hosting & Cloud Infrastructure:**
  - Linode for cloud infrastructure
